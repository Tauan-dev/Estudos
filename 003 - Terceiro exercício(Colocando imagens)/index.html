Vamos colocar algumas imagens no site

para colocar uma imagem no site, ela tem que estar dentro da pasta do index

a tag img é utilizada para por imagens no site,onde ela tem dois parametros, onde a src vai ser a origem da imagem,de onde ela veio(nome do documento) e a alt é um texto alternativo, uma "explicação do que é a imagem"

podemos abrir através de pastas dentro do seu documento
